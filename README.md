## Project Overview
This repository contains my complete Digital Literacy Portfolio. Serving in the role of a Student Digital Ambassador, the objective of this project is to create resources that help peers navigate the digital world safely, responsibly, and professionally. The repository is organized into five distinct tasks, covering visual design, digital portfolios, coding platforms, email etiquette, and cybercrime awareness.
